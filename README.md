# 🧠 Handwritten Digit Classification using CNN

This project implements a Convolutional Neural Network (CNN) to classify handwritten digits (0–9) using the MNIST dataset.

## 🚀 Overview

* Built a CNN model using TensorFlow/Keras
* Achieved high accuracy on validation data
* Applied proper data preprocessing and normalization
* Used train-validation split to avoid data leakage

## 🧱 Tech Stack

* Python
* TensorFlow / Keras
* NumPy, Pandas
* Matplotlib

## 📊 Model Architecture

* Conv2D (32 filters, 3x3)
* MaxPooling
* Flatten
* Dense (128 neurons)
* Dropout (0.5)
* Output layer (Softmax)
